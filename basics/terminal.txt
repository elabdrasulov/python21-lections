ls (list)   - показывает все файлы папки внутри текущей директории
ls test - показывает файлы и папки внутри папки 'test'
ls -a   - показывает также скрытые файлы и папки
ls -l   - показывает более подробную информацию
ls -la  - показывает более подробную информацию + скрытые файлы

cd (change directory) - переходит в домашнюю папку (~)
cd test - переходит в папку test
cd .    - переходит в текущую папку (остается в ней же)
cd ..   - переходит в предыдущую папку
cd ../.. - переходит на две директории назад
cd -    - ctrl+z как в fl
cd test/test2 - переходит в папку test2, которая находится в папке test

mkdir   - создает папку test
mkdir test/test2 - создает папку test2 внутри test (если эта папка существует)
mkdir test1 test2 - создает папки test1 и test2

touch test.txt - создает файл test.txt
touch test/test.txt - создает файл test.txt внутри папки test
touch test1.txt test2.txt - создает файлы test1.txt и test2.txt
touch test{1,2}.txt - создает файлы test1.txt и test2.txt

rm test     - удаляет файл test (если это не папка)
rm -rf test - удаляет файл или папку ср всем содержимым (удаляет это все безвозвратно и насильно)
rm - rf /   - удаляет корневую папку компьютера и стирает все данные на компе (не писать никогда)

pwd (print working directory) - показывает текущее местоположение

cat test.txt - показывает содержимое файла

nano test.txt - открывает редактор кода nano в терминале
code test.txt - открывает text.txt в редакторе кода vscode

sudo - права суперюзера (superuser do)
sudo apt install git - установить систему git
(apt - системный установщик пакетов)
sudo snap install code --classic    - установить программу vscode(snap)
(snap - установщик пакетов (приложений))
